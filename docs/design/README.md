# Approved visual direction

`approved-game-concept.png` is the visual source of truth for the PWA: a late-night card table with charcoal felt, warm ivory cards and type, vermilion actions, lime connection signals, condensed display lettering, and strong one-handed controls.

The production UI translates that concept into accessible HTML/CSS rather than reproducing mockup text. The ornate card back, table texture, and icon crops are project-local raster assets; ranks, suits, status, and controls remain code-native for accuracy and responsiveness.
